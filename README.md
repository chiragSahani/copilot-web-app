## ⚡ Quick Start

```notin
1. Clone & Install
   > git clone https://github.com/chiragSahani/copilot-web-app.git
   > cd copilot-web-app
   > npm install

2. Set up Environment
   > .env.local
   > OPENAI_API_KEY=your_openai_key_here

3. Start Development
   > npm run dev
   > Visit http://localhost:3000
```
